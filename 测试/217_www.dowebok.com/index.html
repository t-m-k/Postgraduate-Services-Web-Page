Scrollanim

Scrollanim 是一款用 CSS3 和 JavaScript 实现的滚动动画库，它可以用在任何地方。

想看更多效果？请往下滚动。

使用简单

Scrollanim 有很多内置的 CSS3 动画，你可以添加到元素的 data-kui-anim 属性上。

<div data-kui-anim="fadeIn">

欢迎来到 www.dowebok.com

</div>

就是这样，很酷，是吧？

高级模式

此外，还可以使用 JavaScript 来添加动画：

kissuiScrollAnim.add(element, {

'in': 'fadeIn'

});

Scrolanim 支持元素所有可用的位置，还可以使用复合事件名称（如："center middle"）。

CSS3 动画

Scrolanim 有很多 CSS3 动画可以使用，没有 JavaScript 动画，使性能更好。

可以运行在 PC 和移动设备。

无限动画

Scrolanim 的动画名接受 infinite 参数，可让动画一直重复播放。
